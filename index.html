我的向往

掬 一杯清茗，捋一抹馨香，品着记忆里留下的温醇，思念着岁月里擦不掉的青春。一直随着时光的脚步，不停歇的走着、走着，偶尔的欣赏着身边不断退后的风景，走 走又停停。曾经，向往过许多美好的东西：谈一场长久的恋爱，考一所梦寐以求的大学，寻一份喜欢的工作？？？？？？，不知何时，现实这个字眼真的变的现实起 来了，好像是理想中的蛀虫，使得理想支离破碎。依然心存憧憬和渴望，执着旧时的追求，那些渴望依旧在时光的隧道里无尽的延伸着。

掌握技能

html

翻译为 超文本标记语言通过标记符号标记要显示的网页中的各个部分 由浏览器解释执行的语言如文字如何处理，画面如何安排，图片如何显示

css

css翻译为层叠样式表，用于控制网页样式并允许样样式信息网页内容分离的一种标记型语言 可以直接由浏览器解释执行的 css是一种定义式结构

浮 动

需要改变排布方式的时候 分类 左幅 float：left； 右浮 float right； 浮动的元素会脱离文档流 浮动的停止 碰到父元素的边

定位

特点：脱离文档流 何时用:元素需要层叠的时候用 1. 找父元素 position：relative； 2. 找子元素 position：absolute；

动画
